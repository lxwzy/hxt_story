简介
通过运用html5技术，用第一手的视频和图片，再现了胡旭涛同学的追梦路。通过展现胡旭涛同学追梦路上的艰辛与坎坷，以及最后的虽败犹荣。宣扬了胡旭涛同学身上的不平凡的坚毅，平凡的感动。
现在，您可以打开index.html文件开始欣赏
